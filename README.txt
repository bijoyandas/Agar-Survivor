This is simple 2d surviving game.
Every user starts with 20 lives.
Whenever the user hits an enemy(red squares), his health decreases. Number of enemies keeps on increasing over time.
Players have 2 weapons. Bullet attack, in which the player either clicks or presses buttons to fire bullets.
The other is blast attack which can be fired using Space bar.

Controls:
Mouse:

Drag mouse to move player
Left mouse button for shooting.

Keyboard:
W - Player up
S - Player down
A - Player left
D - Player right

Up arrow - Fire bullet
Space bar - Blast attack
